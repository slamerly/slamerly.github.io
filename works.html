Masquerade

Subject:

Masquerade is my graduation project, developed as a group.
It's a puzzle-adventure game set in a deceptive theater. You find yourself performing in a well-known play while trying to save your wife.
I had the privilege of being both, Producer and Lead Programmer. I managed the migration of the project from GitHub to Perforce and integrated Wwise at the request of the Sound Designers.
This game features two distinct environement that function differently: the theater's stage and the backstage area. For the smooth operation of the game, I needed to create specefic loading system for these two game zones that would work seamlessly with progress system.

Worked on:

  • Project & Team Management
  • Product Documentation
  • Sublevel Loading Systems
  • Progress System (Save and Checkpoints)
  • UI (Main Menu, Debug Menu and Pause Menu)
  • Tools (Lights' Management)
  • WWISE Integration

Details:

Release Date: May 2025
Language: English, French
Roles: Producer, Lead Programmer, Programmer
Production Time: 11 months
Engine: Unreal
Platform: PC

Squeaky Clean

Subject:

Squeaky Clean is a group project created during the Global Game Jam 2025, with the theme "Bubbles".
The goal of the game is to satisfy the maximum number of machines before the timer runs out. Each machine has its own timer, and you need to satisfy them by bringing them the objects they desire.
But becarefull! If you bring them the wrong object, they'll become unsitisfied and reduce the time you have to satisfy them. Also, watch out for the big plastic ducks; they're easy targets that will come after you!

Worked on:

  • Interaction System
  • Game Rules - Score & Timer
  • Machine Spawner
  • Machine Behavior
  • Machine Timers
  • End Game - Scoreboard

Details:

Release Date: January 2025
Language: English
Roles: Programmer
Production Time: 48 hours
Engine: Unreal
Platform: PC

Bas Haut Bab

Subject:

Bas Haut Bab was a group project created at the request of Montpellier Town Hall for the "Coeur De Ville en Lumière" Festival of 2023. We had specific constraints, including ensuring accessibility to a wide audience and making the game as fun to play as it was to watch.
We also had to consider visual contraints, as the game was projected onto the facade of a historic building.
This is a cooperative game where the aim is to complete as many sequences as you can before the timer runs out. You need to exchange gems and bring them to the correct mask to earn game points. At the end, a scoreboard allows you to compare your performance with other duos.

Worked on:

  • Game and Score Systems
  • Creation and Management of Obstacles
  • Gem spawner
  • Gem physics
  • Tutorial

Details:

Release Date: November 2023
Language: French
Roles: Programmer
Production Time: 1 month
Engine: Unreal
Platform: PC

Cot Cot Party

Subject:

Cot Cot Party is a personal project I developed solo within a 1-month timeframe, centered around the theme of "Fun".
The goal is to achieve the highest score at the end time limit by firing into a swarm of robot chickens. This game was created as part of a 2nd year exercise where, each month, we had to make a game with Unity based on a specific theme durint our free time.
I was inspired by arena FPS games like Unreal Tournament.
